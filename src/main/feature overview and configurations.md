# Overview features and configurations
- --

## Ticketsystem

The Ticketsystem allows server administrators to create tickets and combined ticket panels.

### Panel configurations

- Panel title
- Panel description
- Is subpanel?
- Available options (tickets or subpanels)
- Panel's button color (subpanel only)
- Panel's button label (subpanel only)

### Ticket configurations

- Message title
- Message description
- Ticket/channel name: `x-y`
    - **x**:
        - "ticket" *(default)*
        - custom text
    - **y**:
        - number *(default, auto increment)*
        - user's global name
- Moderation roles / users
- Channel permissions
    - Defaults set to True:
        - Send text message
        - Attach files
        - Read message history
- Save transcript?
- (Sub)panel's button color
- (Sub)panel's button label
